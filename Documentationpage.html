HTML

HTML stands for HyperText Markup Language and is the standard markup language for creating web pages and web applications.

It uses various elements and tags to structure the content on a webpage, such as headings, paragraphs, lists, and more.

<p>This is a paragraph element.</p> <h1>This is a heading element.</h1>

CSS

CSS stands for Cascading Style Sheets and is used to control the presentation and layout of web pages.

It allows you to apply styles to HTML elements, such as changing colors, fonts, margins, and more.

body { background-color: #f2f2f2; } h1 { color: #007bff; }

JavaScript

JavaScript is a high-level, interpreted programming language that adds interactivity and dynamic content to web pages.

It is commonly used for client-side scripting, allowing you to create interactive web elements and respond to user actions.

Media Queries

Media queries are a feature in CSS that allows you to apply different styles based on the characteristics of the user's device or screen size.

It is commonly used for creating responsive web designs that adapt to various devices, such as laptops, tablets, pc, desktops, and smartphones.

@media (max-width: 768px) { /* Styles for mobile devices */ } @media (min-width: 768px) and (max-width: 1024px) { /* Styles for tablets */ }
